Welcome to EnterpriseClaw

Deterministic Agentic Framework

This wizard will guide you through setting up your personal AI assistant in a few quick steps. You'll need:

  1. A Google AI Studio API Key — powers the intelligence (free tier available)
  2. A Telegram Bot Token — connects EnterpriseClaw to your Telegram
  3. Your Telegram Chat ID — restricts access to only you
  4. Google Workspace credentials — optional, for email & calendar

Each step takes about 1–2 minutes. Let's begin. ⚡

Google API Key

This key powers EnterpriseClaw's intelligence via the Gemini 2.5 Flash model.

  1. Go to Google AI Studio → API Keys
  2. Click "Create API Key"
  3. Copy the key and paste it below

Telegram Bot Token

EnterpriseClaw lives inside Telegram. You'll create a bot via BotFather.

  1. Open Telegram and search for @BotFather
  2. Send the command /newbot
  3. Choose a name (e.g., "EnterpriseClaw") and a username (e.g., "my_claw_bot")
  4. BotFather will reply with your bot token — copy it below

Telegram Chat ID

This restricts who can talk to EnterpriseClaw. Only your chat ID will be allowed.

  1. Open Telegram and search for @userinfobot
  2. Send it any message — it will reply with your Chat ID (a number)
  3. Paste that number below

Multiple users? Separate IDs with commas: 123,456,789
